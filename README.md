# Osaifu_Installer
FxxkDocomoAppsでお財布ケータイ機能が消えた際に使用してください。

# How to Use
！！ADBのインストールが必要です！！
実行して復元ボタンをクリックすることで、おサイフケータイ機能を復元できます。
削除する際は削除ボタンをクリックしてください。
